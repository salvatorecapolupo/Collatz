Analizzatore di Collatz

Il sistema calcola la traiettoria di un numero $$n$$ secondo la funzione:

$$f(n) = \begin{cases} n/2 & \text{se } n \equiv 0 \pmod{2} \\ 3n+1 & \text{se } n \equiv 1 \pmod{2} \end{cases}$$
Prova numeri curiosi:
Passaggi: -
Picco Massimo: -
Perché alcuni numeri sono "ostici"?

I numeri che richiedono più tempo a scendere sono spesso vicini alle potenze di 2, forzando la sequenza a salire ripetutamente. Una formula generatrice di casi complessi è:

$$n_{\text{ostico}}(k) = 2^k - 1$$

Questi numeri (Mersenne) iniziano con una lunga serie di bit a 1, ritardando la divisione per 2.

Implementazione in C++ dei numeri di Collatz


#include <iostream>
using namespace std;

int main() {
    // Usiamo long long per la robustezza contro numeri grandi
    long long n;
    int passaggi = 0;

    cout << "--- COLLATZ EXPLORER ---" << endl;
    cout << "Inserisci un numero intero positivo: ";
    
    // Check di robustezza: l'input e' un numero? E' positivo?
    if (!(cin >> n) || n <= 0) {
        cout << "ERRORE: Devi inserire un numero intero maggiore di zero." << endl;
        return 1; // Termina con codice di errore
    }

    cout << "Sequenza: " << n;

    while (n != 1) {
        if (n % 2 == 0) {
            n = n / 2;
        } else {
            n = 3 * n + 1;
        }
        cout << " -> " << n;
        passaggi++;
    }

    cout << "\n\nAnalisi completata in " << passaggi << " passaggi." << endl;

    return 0;
}
